# 🎰 Robô das Taxas — Expert Financeiro v2.0 🎰

O terminal definitivo para investidores profissionais da B3. Uma aplicação de alta performance desenvolvida em React para análise fundamentalista e técnica em tempo real. 🚀💎

## 🚀 Novidades da Versão 2.0 (Expert Financeiro)

### 💎 Calculadora de Valuation Institutional
Chega de preencher tabelas manualmente. O robô agora faz o trabalho duro:
- **Auto-Intelligence**: O sistema captura sozinho **LPA, VPA e Dividendos** via API Brapi e injeta nos modelos.
- **Layout Profissional**: Visualização em 3 colunas (Preço Atual | Preço Justo | Upside) — Padrão BRSR3.
- **Modelos Integrados**: Benjamin Graham, Décio Bazin e Peter Lynch em uma única tela.

### 📈 Gráficos Bloomberg-Style
- **Interatividade Total**: Crosshair (linha de guia) que segue o mouse horizontalmente.
- **Dashboard Resiliente**: Blindagem contra erros de dados históricos.
- **Performance**: Renderização rápida com Google Charts e suporte a múltiplos períodos (1D a Max).

### 🔍 Radar de Notícias Inteligente
- Filtro dinâmico que foca exclusivamente em notícias da empresa pesquisada.
- Eliminamos o ruído de simuladores para focar em fatos relevantes.

## 🛠️ Tecnologias de Elite
- **Core:** React.js com Hooks para gestão de estado em tempo real.
- **Data Engine:** Integração com **Brapi.dev** para cotações e fundamentos oficiais.
- **UI/UX:** Glassmorphism, CSS Variables e Design responsivo premium.

## 🚀 Como Iniciar
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Inicie o terminal: `npm start`.
4. Digite o Ticker (ex: PETR4) e deixe o robô trabalhar.
